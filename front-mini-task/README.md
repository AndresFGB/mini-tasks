# Task Manager — Frontend (React)

Frontend de la prueba técnica: autenticación (login/register), dashboard de tareas con filtros, CRUD de tareas mediante modal, consumo de API Laravel con JWT (Bearer token), y rutas protegidas.

---

## Requisitos

- Node.js 18+ (recomendado)
- npm / pnpm / yarn
- Backend corriendo (Laravel API)

---

## Instalación

1) Clona el repo y entra al proyecto:


git clone
cd 
Instala dependencias:

npm install
Crea el archivo .env en la raíz:

npm run dev
